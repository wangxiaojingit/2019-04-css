css 第二天笔记

一个网页的基本组成所需要的语言

  • html：它是一个超文本标记语言，它不仅单纯的表示文本，还可以表示音频、视频、图片等。（骨架）

  • css：是一个层叠样式表，它的作用主要是美化页面（主要是样式）

  • js：它是一门轻量级的编程语言，它主要是用来写业务逻辑和一些行为动态，比如点击事件等。（行为和动态）

前端小概念

网页：一个html页面。（书中的每一页）
网站：由多个网页组成 （书），像京东、淘宝。

网页的基本结构

<!-- 声明文档类型，如果不写的话，会触发怪异模式 -->

<!DOCTYPE html>

<html>

    <head>

        <!--charset="utf-8" 国际通用编码，不写会乱码  -->

       <meta charset="utf-8">

       <!-- 描述，给搜索引擎看的 -->

       <meta name="description" content="描述字">

       <!-- 关键字，给搜索引擎看的 -->

       <meta name="keywords" content="关键词">

       <title>网页的标题</title>

    </head>

    <body>

          可视区

    </body>

</html>

常用的标签

1、 段落：p
<p>习近平指出，世界正处于百年未有之大变局。</p>
2、标题：h1-h6

快速生成六级标题：h{标题}*6 +tab
h1到h6 字体从大到小逐次递减，粗细亦如此。从优化上来说，h1是最重要的，一个页面中通常只出现一次，一般我们页面中的logo会用h1包裹，就是为了搜索引擎优化。

          <h1>标题1</h1>

          <h2>标题2</h2>

          <h3>标题3</h3>

          <h4>标题4</h4>

          <h5>标题5</h5>

          <h6>标题6</h6>
a标签：跳转链接

基础用法：

 <!-- 

       href:页面跳转的网址

       target：

          1、_self 在当前窗口打开

          2、_blank 在新窗口打开

     -->



    <a href="http://www.baidu.com" target="_self">百度</a>

href 的几个用途

  • 1、如果href里面放的是“#”，点击的时候会跳转到页面的顶部

  • 2、如果是“javascript:;”禁止跳转

  • 3、做锚点跳转，href里面放的是元素id（“#” 不要忘记）

css引入的几种方式

1、行内式：给谁增加样式，就在谁的开始标签里面写样式，格式例如：style=“color：green”;

 <div style="color:green;font-size:36px;">我是div</div>

2、内嵌式：在head头部里面通过style标签 然后给谁写样式，就选中谁。格式例如：

    <style type="text/css">

     div{

         /* 字体颜色 */

         color:green;

         /* 字体大小 */

         font-size:38px;

         /* 加粗 */

         font-weight: 600;

     }

    </style>

3、外链式：通过link标签把写好的样式表和当前的html页面进行关联。通常在项目中使用。

 <link rel="stylesheet" href="index.css">

4、导入式: 这个不常用，它是一条命令语句。

    <style>

    @import "index.css"

    </style>

在css样式中，代码也是从上往下执行，如果在权重一样的情况下，后面的代码会覆盖前面的代码样式。

面试题：i和em ，都是斜体， b和strong 都是粗体，那它们有什么区别呢？

i是斜体，b是粗体，它们只是表象上达到了一定的样式，但是em和strong除了样式上具备，语气更加的强烈，更有利于搜索引擎的优化。

三大列表

1、有序列表：ol，ol里面紧挨着li，li里面可以嵌套其它标签。有序列表默认前面会出现数字。

 <ol>

       <li>上班</li>

       <li>吃饭</li>

       <li>打豆豆</li>

 </ol>

2、无序列表：ul，ul里面紧挨着的是li，li里面可以嵌套其它的标签。无序列表默认前面是一个小圆点。

<ul>

        <li>上班</li>

        <li>吃饭</li>

        <li>打豆豆</li>

</ul>

3、自定义列表：dl，dl里面是dt和dd。dt代表的是列表的标题，dd代表的是对标题的描述。（每一个小项）

<dl>

        <dt>早餐</dt>

        <dd>牛奶</dd>

        <dd>豆浆</dd>

        <dd>油条</dd>

        <dt>午餐</dt>

        <dd>肉盒子</dd>

        <dd>麻辣烫</dd>

        <dt>晚餐</dt>

        <dd>鸡蛋</dd>

        <dd>牛奶</dd>

    </dl>
