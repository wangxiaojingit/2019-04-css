第三天css笔记

块级元素的特点：

  • 很霸道，独占一行；

  • 设置宽、高可以起作用；

  • 排列方式：从上到下；

行内元素的特点：

  • 可以和行内元素或者行内块共占一行

  • 设置宽、高不起作用，它的大小是由本身的内容所决定的；

  • 排列方式：从左到右；

行内块元素的特点：

  • 可以和行内元素或者行内块共占一行；

  • 设置宽、高起作用；

  • 排列方式：从左到右；

汇总标签：

  • 段落：p

  • 标题：h1-h6

  • 超链接：a

  • 图片：img

  • 大盒子标签：div

  • 小盒子标签：span

  • 格式化标签：

    • 加粗：b、strong

    • 斜体： i、em

    • 删除线：del

    • 变大：big

    • 变小：small

  • iframe标签

  • 三大列表：ol、ul、li、dl、dt、dd

块级元素：

div、p、h1-h6、ol、ul、li、dl、dt、dd

行内元素：

a、span、b、strong、i、em、del、big、small

行内块：

img

display的几个值

  • display:inline-block; 转换成行内块

  • display：inline； 转换成行内元素

  • display：block; 转换成块级元素

  • display：none；让一个元素消失

面试题：让一个元素消失的几种方法：

  • display：none；

  • visibility:hidden; 隐藏

  • opacity:0; 透明度为0；

  • width、height 为0；

  • 利用z-index 进行层级覆盖；

  • margin 负值或者定位到屏幕的一侧；

display：none 它是彻底的消失了，visibility:hidden 和opacity：0 它虽然看不见了，但是位置还在。

css 一些常用的样式

1、表示一个元素的宽（width）、高（height），

   <style>

     div{

         width:200px;

         height:200px;

         background-color:red;

     }

    </style>

2、表示背景色：background-color：green；

    div{

         width:200px;

         height:200px;

         background-color:red;

     }

3、控制字体大小：font-size:38px;
4、文字颜色： color:red;
5、字体：font-family，中文的话，会加双引用，英文不加双引号。

font-family:"宋体","Times New Roman",Georgia,Serif;

6、消除页面中所有元素自带的margin值和padding值。

*表示通配符选择器，选中所有的元素，把他们默认带的margin值和padding值都归为0;

    <style>



      *{

          margin:0;padding:0;

       }  

     div{

         width:200px;

         height:200px;

         background-color:red;

         font-size:38px;

         color:cadetblue;

         font-family: "微软雅黑"

     }

    </style>

7、清除a标签自带的下划线

     a{

         text-decoration:none;

     }

8、清除ol、ul列表默认自带的小圆点或者数字

     ul,ol{

         list-style:none;

     }

9、块级元素在一个盒子中居中：margin:0 auto;

  #box1{

        width:200px;

        height:200px;

        background:chartreuse;

        /* 第一个值：上下

           第二个值：左右

         */

        margin:100px auto;

    }

10、控制文字的粗细：font-weight：value；

  • 正常：normal ;如果是h2标题，不让加粗，就可以写normal；也可以直接写成400；

  • 加粗：bold;（700）；

<style>

    p{

        font-weight:bold;

        font-size:28px;

    }

    h2{

        font-weight: normal;

    }

    </style>

11、文字对齐方式：text-align：value；

  • 居左：left；（默认）

  • 居中：center；

  • 居右：right；

   h2{     

        /* 

          居左：left 默认

          居中：center

          居右：right

         */

        text-align:left;

    }

12、文字缩进：text-indent:value;

  • 2em:代表缩进两个字的距离；

  • 还可以跟具体的像素值，例如：text-indent:10px;
