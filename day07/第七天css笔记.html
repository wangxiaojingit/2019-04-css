1、.box>li:nth-child(2n+1){}，代表的意思：选择所有儿子元素中的第几个

<html>

<head>

    <meta charset='UTF-8'>

    <meta name='keywords' content='关键词'>

    <meta name='description' content='描述的文字'>

    <title>c3选择器</title>

    <style>

        /* 

        odd 奇数

        even 偶数

         */



    /* .box>li:nth-child(2n+1){

        background:green;

    } */

    .box>li:first-child{

        background:green;

    }

    .box>li:last-child{

        background:blanchedalmond;

    }

    /* 倒数 */

    .box>li:nth-last-child(2){

         background:blue;

    }

    </style>

</head>

<body>

    <ul class="box">

        <li>li1</li>

        <li>li2</li>

        <li>li3</li>

        <li>li4</li>

        <li>li5</li>

        <li>li6</li>

    </ul>

</body>

</html>

定位

  • 默认的是static

  • 相对定位：relative

  • 绝对定位：absolute;

  • 固定定位：fixed；

相对定位

让谁相对定位，就给谁添加position：relative,方向值：left、right、bottom、top。

相对定位的特点：

  • 1、相对自身

  • 2、没有脱离文档流

  • 3、层级高于普通文档流

  • 4、最重要的，给绝对定位做参照物。

  • 5、如果同时把四个方向的值都写了，默认会是left 和top

绝对定位

使用：想要谁绝对定位，就给谁添加position:absolute;但是一定要设置参照物，给它的参照物设置,position:relative;如果它的参照物本身就具备position：relative,或者是position:absolute,或者是position：fixed，三者之一都能满足这个参照物条件。
绝对定位的特点：

  • 1、脱离文档流

  • 2、层级高于普通文档流

  • 3、需要注意的，在设置参照物的时候，这个参照物不能是同级，必须是上级元素（前辈元素都可以）。

  • 4、浏览器在寻找参照物的时候有个规则“就近原则”。先找父级元素，如果父级元素中满足参照物的条件，那就是父级元素，如果父级不满足，就继续向父级的父级继续查找，如果都没有，那参照物就是body

  • 5、定位的元素如果不设置宽度，它的宽度是内容的大小，如果用百分比设置宽度，它是按照参照物的宽度，不是父级。

固定定位：

想让谁固定定位，就给谁设置position:fixed;
固定定位的参照物就是浏览器窗口。
