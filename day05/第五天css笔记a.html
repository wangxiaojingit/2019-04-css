选择性别：<label for="man">男</label><input id="man" type="radio" name="sex">  <label for="woman">女</label> <input type="radio" name="sex" id="woman">

盒模型

盒模型是一种写页面的思维方式，每个元素都可以看成一个盒子，盒子的内容就是content，内容到盒子边框的距离就是padding（内填充），盒子的边框就是border；盒子与盒子之间的距离，就是margin；

用手机盒子来举例，手机就是content，手机盒子中的白色泡沫就是padding，手机盒子的厚度就是border，多个手机盒子之间的距离就是margin；

面试题：盒模型主要包括哪几部分？

  • content

  • padding

  • border

  • margin

盒子的总大小计算方式：

盒子总的width=content的宽+2*padding+2*border;
盒子总的height=content的高+2*padding+2*border;

盒模型的分类：

  • 标准盒模型：box-sizing:content-box;（默认）

  • ie盒模型：box-sizing:border-box;(在写页面的时候通常用这个)。

标准盒模型，咱们代码中所写的那个宽度,或者高度，就变成了content部分的宽、高，总盒子的大小就是content+2*padding+2*border；

ie盒模型，咱们代码中所写width和高度就是整个盒子的总大小，content部分会自动进行计算。

margin值：

  • margin-left:距左；

  • margin-right:距右；

  • margin-top:距上；

  • margin-bottom：距下；

.box1{

        /* 上右下左，按照顺时针排列 */

        /* margin:10px 100px 200px 300px; */

        /* 上、左右、下 */

        /* margin:10px 100px 200px; */

        /* 上下、左右 */

        margin:10px 100px;

        /* 上下左右四个方向一致 */

        margin:10px;



    }

面试题margin值穿透的问题：

当一个大盒子包含一个小盒子，大盒子没有设置border-top或者padding-top的时候，我们给小盒子设置margin-top，大盒子会跟着小盒子一起向下移动，这个现象就叫做margin值穿透。

解决办法：

  • 1、给父盒子添加一个ovflow:hidden;

  • 2、给父盒子设置一个border-top:1px solid transparent;

  • 3、margin-top既然存在一定的问题，我们可以改用padding值，给父盒子设置padding-top同样可以达到一样的效果。

margin值合并问题

当两个兄弟元素，例如两个div元素，一个是box1，一个是box2，他俩是垂直排列，当我给box1设置一个margin-bottom：30px; 同时给box2设置了一个margin-top：50px;这两个盒子之间的距离并不是我们所想的30+50，而是选择最大的那个值。

解决办法，避免上面的情况，可以给一个盒子设置就行了，或者是设置成都是同方向的就可以了。
